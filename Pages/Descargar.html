Descargar Jakecherrys Launcher

Para instalar el launcher Jakecherrys necesitas Python 3.12.2

Una vez instalado Python necesitaras ejecutar los siguientes comandos en CMD:

  • pip install minecraft-launcher-lib==6.4
  • pip install customtkinter==5.2.2
  • pip install pillow==10.2.0

Una vez terminados esos pasos descargaras como ZIP los archivos de este repositorio de Github

Repositorio de Github

Una vez terminados esos pasos extraes el archivo ZIP y accedes a la carpeta de la ultima version, aparecera como "Jakecherrys [ID de la version]"

Despues de eso puedes hacer un acceso directo al archivo "Main.py" que es el que te deja acceder al launcher para un acceso rapido.

Gracias por leer y utilizar nuestro launcher OpenSource.

Si tienes problemas con la instalacion puedes guiarte con la guia de instalacion de el canal de Youtube Oficial.
